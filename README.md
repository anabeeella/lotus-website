# Lotus Website

This is the website for Lotus, a social media marketing agency.

## Technologies Used

- React
- Chakra UI
- Vite
- React Router
- React Icons
- React Scroll
- React Typed

## Features

- Responsive design
- Smooth scroll
- Typed text
- Scroll animation
- Scroll animation

## Installation

1. Clone the repository
2. Install dependencies
3. Run the development server

    ```bash
    npm install
    npm run dev
    ```
